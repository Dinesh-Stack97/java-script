console.error():

used to log error messages to the console .Useful in testing of code.By default the error message will be highlighted with the red color.

console.error('This is a simple error');

console.log():

y Mainly used to log(print) the otput to the console.we can put any type inside the log(), be it a string , array ,object, boolean etc....

console.log('abc'); console.log(1); console.log(true); console.log(null); console.log(undefined); console.log([1, 2, 3, 4]);

console.warn()

used to log warning messages to the console.By default the warning messages will be highlighted by yellow color.

console.warn('This is a simple warning error');

console.time()

whenever we want to know the amount of time spend by a block or a function , we can make use of the time() and the timeEnd() methods provided by the js console .

console.time('abc');

console.table():

This method allows us to generate a table inside the console.The input must be an array or an object which will be shown as the table..

console.table({'a' :1, 'b' :2});

These methods of the console object allows us to group contents in a seperate block, which will be indented .Just like the time() snd the timeend() they accept labels and vslues
